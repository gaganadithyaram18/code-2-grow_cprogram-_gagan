## Description
This C program allows users to input employee salary data and perform various analyses such as total, average, highest, and lowest salary checks.  
It also provides sorting functionality and a count of employees above and below average salary.

---

## Build Commands
### Windows (MinGW):
```bash
mkdir build
gcc -std=c11 -Wall -Wextra -O2 main.c -o build/program.exe
```

### Linux/macOS:
```bash
mkdir -p build
gcc -std=c11 -Wall -Wextra -O2 main.c -o build/program
```

---

## Run Commands
### Windows:
```bash
build\program.exe
```

### Linux/macOS:
```bash
./build/program
```
